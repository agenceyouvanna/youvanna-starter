name: Always continue all phases
description: TOUJOURS continuer toutes les phases sans s'arrêter, ou demander explicitement si bloqué
type: feedback
---

TOUJOURS continuer toutes les phases d'un projet sans s'arrêter. Ne JAMAIS attendre que l'utilisateur dise de continuer.
**Why:** L'utilisateur veut que le travail soit complet. S'arrêter entre les phases donne l'impression d'un travail inachevé et fait perdre du temps.
**How to apply:** Quand une phase est terminée, enchaîner immédiatement sur la suivante. Si quelque chose bloque (clé API manquante, spec incomplète), demander explicitement au lieu d'attendre silencieusement. "Lance tout et ne t'arrête pas" est la règle par défaut.
